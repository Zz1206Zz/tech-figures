# 赛博科技风个人简历网站 · 设计规范

> 版本：v1.0  
> 日期：2026-07-06  
> 设计目标：基于现有家教简历网站，打造一个具有高级感、科技感的仪表盘式个人品牌网站

---

## 一、项目概述

### 1.1 项目背景
基于现有的「广州增城家教 - 曾学长」个人简历网站，重新设计为赛博科技风格的仪表盘式网站，突出「AI赋能教学」和「自主学习能力」的核心亮点。

### 1.2 核心目标
- **建立专业形象**：通过高级感的设计和科技感的视觉语言，塑造专业、前沿的个人品牌
- **突出AI能力**：新增「AI自主建站」核心优势展示，彰显创造力和学习力
- **提升品牌记忆点**：独特的仪表盘布局和冰蓝冷白配色，让访客过目不忘

### 1.3 目标受众
- 学生家长（决策者）
- 有辅导需求的学生
- 关注AI教育的人群

## 二、设计风格

### 2.1 风格定位
**赛博科技风 · 冰蓝冷白色调 · 仪表盘式布局**

整体感觉像一个「AI学习系统控制台」，冷峻、高级、充满未来感，同时保持信息的清晰可读性。

### 2.2 配色方案

| 层级 | 变量名 | 色值 | 用途 |
|------|--------|------|------|
| 主背景 | `--bg-primary` | `#05070d` | 页面底色，深邃 |
| 卡片背景 | `--bg-card` | `#0d1220` | 卡片默认底色 |
| 卡片悬浮 | `--bg-card-hover` | `#111827` | 卡片悬停底色 |
| 主色调 | `--accent-primary` | `#00d4ff` | 冰蓝色，主强调色、图标、边框高亮 |
| 辅助色 | `--accent-secondary` | `#7c3aed` | 紫色，渐变点缀、次要强调 |
| 成功色 | `--accent-success` | `#22d3ee` | 青色，数据正向、在线状态 |
| 警告色 | `--accent-warn` | `#f59e0b` | 琥珀金，价格、重要数据 |
| 主文字 | `--text-primary` | `#e2e8f0` | 浅灰蓝，正文文字 |
| 次文字 | `--text-secondary` | `#64748b` | 冷灰色，次要文字、说明 |
| 边框 | `--border-color` | `#1e293b` | 深灰蓝，卡片边框、分割线 |
| 边框高亮 | `--border-active` | `#00d4ff` | 冰蓝色，激活/悬停边框 |

### 2.3 字体方案

- **英文/数字**：`JetBrains Mono` 等宽字体（科技感、代码感）
- **中文标题**：`PingFang SC` / `Microsoft YaHei`，加粗，字间距 `0.02em`
- **中文正文**：`PingFang SC` / `Microsoft YaHei` / `Noto Sans CJK SC`，常规字重
- **字体层级**：
  - 大标题：32-48px / 700
  - 卡片标题：18-20px / 600
  - 正文：14-16px / 400
  - 说明文字：12-13px / 400

### 2.4 视觉元素

1. **网格背景**：淡蓝色细网格线（`opacity: 0.08`），类似电路板/全息投影
2. **光晕效果**：卡片悬停时冰蓝色外发光（`box-shadow: 0 0 20px rgba(0, 212, 255, 0.15)`）
3. **扫描线**：微妙的横向扫描动画（极低透明度，克制使用）
4. **毛玻璃**：重要卡片用 `backdrop-filter: blur(12px)` 增强层次感
5. **渐变边框**：激活状态的卡片使用冰蓝到紫色的渐变边框

## 五、GSAP 动画设计

### 5.1 动画原则

1. **精致克制**：每个动画都有目的，不为动而动
2. **快速干脆**：时长 0.3-0.8s，缓动 `power2.out / power3.out`
3. **滚动触发**：大部分动画由 ScrollTrigger 触发
4. **悬停反馈**：交互元素有明确但微妙的反馈
5. **性能优先**：只动画 `transform` 和 `opacity`，保证 60fps

### 5.2 入场动画（首屏加载）

| 元素 | 动画效果 | 时长 | 延迟 |
|------|---------|------|------|
| 背景网格 | 淡入 + 轻微缩放 | 0.8s | 0s |
| Hero 卡片边框 | 渐变描边绘制 | 1s | 0.2s |
| 主标题文字 | 逐字淡入上移 | 0.6s | 0.4s |
| 状态标签 | 从左滑入 + 渐显（stagger） | 0.4s | 0.6s |
| 打字机效果 | 文字循环打字 | 持续 | 1s |

### 5.3 滚动触发动画（ScrollTrigger）

**通用卡片入场：**
- `y: 20px → 0`
- `opacity: 0 → 1`
- 时长：0.5s
- 缓动：`power2.out`
- 触发时机：`start: "top 85%"`
- 同一行卡片 stagger：0.1s

**数字滚动：**
- 从 0 滚动到目标值
- 时长：1.2s
- 缓动：`power2.out`
- 使用 `gsap.utils.snap` 确保整数

### 5.4 悬停动画

**卡片通用悬停：**
- `translateY(-4px)`
- 边框颜色：深灰 → 冰蓝
- `box-shadow`：冰蓝色外发光（低透明度）
- 时长：0.25s
- 缓动：`power2.out`

**按钮悬停：**
- 轻微放大 1.02 倍
- 光晕增强
- 时长：0.2s

### 5.5 导航栏动画

- 激活项：左侧指示条滑入 + 文字高亮
- 滚动同步：ScrollTrigger 监听滚动位置，自动切换激活状态
- 悬停：背景微亮 + 文字变亮

### 5.6 特殊动画

**AI建站模块 - 代码打字：**
- 代码逐行显示，每行间隔 50ms
- 光标闪烁效果
- 模拟终端输入的感觉

**金句引用：**
- 引号从 1.5 倍缩放到 1 倍
- 文字从模糊到清晰
- 时长：0.8s

### 5.7 性能与可访问性

- 使用 `will-change` 提示浏览器优化
- 只动画 `transform` 和 `opacity`
- 移动端减少动画数量
- 尊重 `prefers-reduced-motion`，自动降级

### 5.8 GSAP 插件清单

| 插件 | 用途 | 必须 |
|------|------|------|
| `gsap/core` | 核心动画库 | ✅ |
| `ScrollTrigger` | 滚动触发 | ✅ |
| `gsap/utils` | 工具函数（数字滚动等） | ✅ |
| `CSSPlugin` | CSS 动画 | ✅ |
| `EasePack` | 缓动函数 | ⭐ 推荐 |

## 七、响应式设计

### 7.1 断点设计

| 断点 | 设备类型 | 布局调整 |
|------|---------|---------|
| `≥ 1200px` | 桌面端 | 左侧导航（宽）+ 多列 Bento Grid |
| `768-1200px` | 平板 | 左侧导航（窄，仅图标）+ 列数减少 |
| `< 768px` | 手机 | 顶部导航（横向/汉堡）+ 单列卡片 |

### 7.2 移动端策略

- 导航移到顶部，可横向滚动或汉堡菜单展开
- Bento Grid 变为单列垂直排列
- 动画数量减少，保证流畅
- 字号、间距适当缩小
